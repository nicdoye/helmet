# Helmed

A simple command line for Kubernetes Helm repos


## Usage

When finished:

```
helm list
helm list --all-versions
helm search <chartname>
helm search <chartname> --all-versions
```

As above but with

```
--repo <my-super-secret-helm-repo>
```
