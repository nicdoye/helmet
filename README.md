# Helmet

A simple command line for Kubernetes Helm repos

## Usage

When finished:

```bash
helmet list
helmet list --all-versions
helmet search <chartname>
helmet search <chartname> --all-versions
```

As above but with

```bash
--repo <my-super-secret-helm-repo>
```
